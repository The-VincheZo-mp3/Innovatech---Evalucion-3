# Backend Microservice

Repositorio backend del proyecto DevOps para Innovatech Chile.

## Objetivo

Implementar una solución contenerizada y automatizada utilizando Docker, GitHub Actions y AWS EC2.

## Tecnologías

- Docker
- Docker Compose
- GitHub Actions
- AWS EC2

## Arquitectura

Frontend → Backend → Database

## Características DevOps

- Contenedorización
- Persistencia con volúmenes
- CI/CD automatizado
- Despliegue en AWS
